# Five-minute demonstration script

1. Open **Estado atual** and explain the scientific boundary.
2. Select the Southwest Iberian Margin and a 90-day window.
3. Show recent events, regional event rate and historical percentile.
4. Open **Memória semelhante** and inspect the five nearest prior windows.
5. Expand one analogue and compare the feature differences.
6. Open **Replay Portugal**, choose a historical cutoff and run the replay.
7. Compare analogue probability with the historical-rate baseline.
8. Open **Qualidade e metodologia** and show source provenance, completeness and warnings.
9. Finish by stating that similarity is not deterministic prediction.
