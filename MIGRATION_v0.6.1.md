# MEMÓRIA v0.6.1 — Correção de autoria e afiliação

Esta revisão corrige uma possível atribuição errada do projeto ao IPMA.

## Factos fixados

- O MEMÓRIA foi criado e é desenvolvido por **Gonçalo Pedro**.
- É um projeto experimental independente.
- Não é um produto, projeto ou iniciativa oficial do IPMA.
- O IPMA é uma fonte pública de dados integrada e a referência oficial para informação sísmica.

## Aplicação

Copie o conteúdo do patch sobre o projeto v0.6.0, reinstale com `python -m pip install --upgrade -e .`, limpe a cache do Streamlit e reinicie a aplicação.

Não é necessário reconstruir Bronze, Silver ou Gold. O histórico anterior do chatbot é limpo automaticamente na primeira execução da v0.6.1.
